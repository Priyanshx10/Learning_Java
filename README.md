# 🌿 Git Workflow

Every problem is solved in its own feature branch.

### Workflow

1. Create a feature branch
2. Solve the problem
3. Commit the solution
4. Push the branch
5. Create a Pull Request
6. Merge into `main`
7. Delete the feature branch

This repository also serves as Git workflow practice.
